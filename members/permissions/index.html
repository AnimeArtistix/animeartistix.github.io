AnimeArtistix group member permissions (Work in progress)

What are permission? Well, permissions are what group members are allowed/permitted to do. This page is for reference.

These are the permissions of group member titles. Click the things on the side to view different title permissions.

Owner permissions

  • Transfer ownership of group to another group member.
  • Kick out members and add new members.
  • Make a new branch, delete or edit an existing branch.
  • Add, remove, or edit a new mascot.
  • Edit member titles.
  • Ask website developer (JeffTheMasterr) to add new stuff to this website.
  • Whatever you want (as long as it is reasonable!!)

Co-owner and Co-co-owner permissions

  • Kick out members and add new members.
  • Make a new branch, delete or edit an existing branch.
  • Add, remove, or edit a new mascot.
  • Edit member titles.
  • Ask website developer (JeffTheMasterr) to add new stuff to this website.

Admin permissions

  • Ask to kick out members or add new members.
  • Ask to make a new branch, or delete or edit an existing branch.
  • Ask to add, remove, or edit a new mascot.
  • Ask to edit member titles.
  • Ask website developer (JeffTheMasterr) to add new stuff to this website.

Member permissions

  • Ask to kick out members or add new members.
  • Ask to make a new branch, or delete or edit an existing branch.
  • Ask to add, remove, or edit a new mascot.
